# Jarir Marketing Company LBO Model (Tadawul: 4190)

An illustrative Leveraged Buyout (LBO) analysis built on Jarir Marketing Company, a publicly traded Saudi retailer, as a practical case study in PE deal process and LBO modeling.

## What's inside

A 5-tab Excel workbook:

1. **Cover** — project overview
2. **Assumptions** — transaction, financing, and operating assumptions (each sourced)
3. **Sources & Uses** — deal financing structure and implied leverage
4. **Operating Model & Debt Schedule** — 5-year projections, debt paydown, and free cash flow build
5. **Returns Analysis** — MOIC, IRR, and a leverage × exit-multiple sensitivity grid

All figures are fully formula-driven and linked across tabs.

## Key assumptions

- Entry: ~15.0x EV/EBITDA (approximating Jarir's current public market trading multiple)
- Financing: 4.0x EBITDA senior term loan, remainder in sponsor equity
- 5-year hold, exit at same 15.0x multiple (base case)

## Base case result

| Metric | Value |
|---|---|
| MOIC | 1.54x |
| IRR | ~9.0% |

## Key takeaway

At Jarir's current market valuation, returns are driven almost entirely by the exit multiple assumption, not leverage flexing entry leverage from 3.0x to 5.0x barely moves the IRR at any given exit multiple. This suggests Jarir, in its current form, is a structurally weak LBO candidate without a credible path to margin expansion, M&A-driven growth, or multiple re-rating at exit.
